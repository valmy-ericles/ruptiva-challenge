# README

Como rodar a aplicação.

No terminal, execute:

* git clone git@github.com:valmy-ericles/ruptiva-challenge.git

Entre na pasta do projeto

* cd ruptiva-challenge

Execute o seguinte comando para instalar as gems
* bundle install

No terminal, execute:
* rails s

# Pré-requisitos
* ruby versão 2.7.1
